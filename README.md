## Resumen
Este repositorio compila mis proyectos integradores desarrollados en Henry. Cada entrega refleja la aplicación de metodologías de ciencia de datos para resolver problemas de negocio reales, desde la ingesta de datos hasta la puesta en producción.

---

## Habilidades Técnicas (Skills)

### **Análisis y Lenguajes**
* **Python:** Pandas, NumPy, Scipy (Análisis estadístico y manipulación de datos).
* **SQL:** PostgreSQL, MySQL (Diseño de esquemas, Joins complejos, optimización de consultas).
* **Estadística:** Análisis descriptivo, test de hipótesis, distribuciones.

### **Machine Learning & Modelado**
* **Supervisado:** Regresiones, Árboles de decisión, Random Forest, XGBoost.
* **No Supervisado:** Clustering (K-Means), reducción de dimensionalidad (PCA).
* **Métricas:** Evaluación de modelos (Accuracy, Precision, Recall, F1-Score, RMSE).

### **Visualización y BI**
* **Herramientas:** Power BI / Tableau (Creación de Dashboards interactivos).
* **Librerías:** Matplotlib, Seaborn, Plotly.

### **Herramientas de Desarrollo**
* **Control de versiones:** Git / GitHub (Flujo de ramas, resolución de conflictos).
* **Entornos:** Jupyter Notebooks, VS Code, Google Colab.
